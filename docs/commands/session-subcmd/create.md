## Usage

```bash
$ session create [-birt] --name string [host]
```

## Parameters


Command parameters:

### `[host]`


The host to connect to the server.

## Arguments


Command arguments:

### `--background, -b`


Do not attach the session when created.

### `--ignore-test, -i`


Ignore connection test.

### `--name, -n`


Name of the session.

### `--raw, -r`


Do not use compressed connection.

### `--token, -t`


Use token.
